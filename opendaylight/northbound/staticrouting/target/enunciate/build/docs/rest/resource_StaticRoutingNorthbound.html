StaticRoutingNorthbound

Static Routing Northbound APIs

Authentication scheme : HTTP Basic
Authentication realm : opendaylight
Transport : HTTP and HTTPS

HTTPS Authentication is disabled by default. Administrator can enable it in tomcat-server.xml after adding a proper keystore / SSL certificate from a trusted authority.
More info : http://tomcat.apache.org/tomcat-7.0-doc/ssl-howto.html#Configuration

The following resources are part of this group:

  • /{containerName}
  • /{containerName}/{name}

/{containerName}

Mount Point: /controller/nb/v2/staticroute/{containerName}

GET

Returns a list of static routes present on the given container

Parameters

name description type default
containerName Name of the Container. The Container name for the base controller is "default". path

Response Body

element: staticRoutes
media types: application/xml
application/json

List of configured static routes on the given container

Status Codes

code description
